# Libreria

Esta es una librería creada como proyecto para el curso de React en Coderhouse.

## Excusas:

No he tenido demasiado tiempo, pero dejo lo mejor posible. Más adelante seguiré actualizando este proyecto para realizar mis prácticas. Me disculpo por no colocar más estilos, pero por el trabajo y un viaje me fue imposible. Pero, ya estoy certficado en CSS por Coderhouse, por si acaso.

## Tecnologías usadas:

- Vite
- React
- React Router DOM
- Firebase
- Toastify
- Bootstrap

## Funcionalidades disponibles:
- Ver listado de libros
- Agregar libros al carrito de compras
- Ver el carrito de compras
- Realizar una orden de compra con los datos del usuario
